Here is a bash script that you can use to install and set up QEMU, libvirt, libvirt-clients, bridge-utils, virtinst, and virt-manager on Ubuntu 22.04:

```shell
#!/bin/bash

sudo apt-get update
sudo apt-get install -y qemu-kvm libvirt-daemon-system libvirt-clients bridge-utils virtinst virt-manager spice-vdagent
sudo usermod -aG libvirt $USER
sudo usermod -aG kvm $USER
sudo systemctl restart libvirtd
sudo systemctl enable libvirtd
sudo systemctl status libvirtd

```

This script installs the necessary packages, adds the current user to the libvirt group, restarts the libvirtd service, and enables it to start on boot. It also checks the status of the libvirtd service to verify that it is running properly.

Drivers for the OS: [VirtIO](https://github.com/virtio-win/virtio-win-pkg-scripts/blob/master/README.md)
